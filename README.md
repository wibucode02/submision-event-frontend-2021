### Faizal Chan.

Magazinc! Website

Magazinc! merupakan salah satu media massa mahasiswa yang bergerak di dalam lingkup Universitas Telkom. Website ini dirancang dengan memperhatikan urgensi atas kebutuhan mahasiswa dalam mendapatkan informasi yang aktual, faktual, dan terpercaya.

Untuk mockup websitenya dapat diakses di sini :
- Halaman utama https://mf-chan.com/Magazinc-Website/
- Halaman berita https://mf-chan.com/Magazinc-Website/berita

Website dirancang dengan mengimplementasikan prinsip mobile-first responsive web design, dikemas dengan brand identity dari Magazinc! Universitas Telkom yang berfokus pada penyampaian berbagai informasi seputar Tri Dharma Perguruan Tinggi, yakni pendidikan, penelitian, dan pengabdian kepada masyarakat.

Teknologi yang digunakan :
- HTML5
- CSS3
- jQuery
- Bootstrap 3
- Font Awesome 5

Hasil analisis perancangan ini sudah terpublikasi di jurnal ilmiah, yang dapat dibaca di sini :
https://doi.org/10.31598/bahasarupa.v4i1.606
